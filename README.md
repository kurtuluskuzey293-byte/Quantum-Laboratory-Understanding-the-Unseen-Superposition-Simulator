# Quantum-Laboratory-Understanding-the-Unseen-Superposition-Simulator
Quantum Laboratory is an interactive virtual lab designed to help students and researchers intuitively understand single-qubit superposition through Bloch sphere visualization. Users can adjust state parameters, observe probability changes, and simulate measurement collapse in real time. Developed by Kuzey Kurtuluş. for intuitive quantum learning.!
Quantum Laboratory is an interactive virtual environment developed to help students and researchers understand single-qubit quantum superposition through geometric intuition and visual exploration.

This simulator provides a simplified but scientifically accurate virtual quantum mechanics laboratory where users can manipulate qubit state parameters (θ, φ), observe probability distributions in real time, simulate wavefunction collapse through measurement, and compare experimental results with theoretical predictions based on the Born Rule.

By combining mathematical structure with geometric visualization via the Bloch Sphere representation, this project aims to make quantum mechanics more intuitive and accessible. The simulator supports learning through visual memory and interactive experimentation, helping users explore the structure of quantum states and the nature of measurement in a clear environment.

This virtual laboratory was developed to make the invisible structure of quantum reality easier to understand through interactive exploration.

Developed by: Kuzey Kurtuluş
Features
Interactive Bloch Sphere visualization
Adjustable quantum state parameters (θ, φ)
Real-time probability updates
Wavefunction component visualization
Measurement simulation (wavefunction collapse)
Experimental vs theoretical probability comparison
Entropy and coherence indicators

How to Run the Simulator
1️⃣ Clone the repository
git clone https://github.com/your-username/quantum-laboratory.git
cd quantum-laboratory
2️⃣ Install dependencies
pip install streamlit numpy plotly
3️⃣ Run the simulator
streamlit run simulator.py

Then open the local address shown in your terminal (usually):

http://localhost:8501

Purpose of the Project

The goal of this project is to provide a clear and accessible virtual quantum laboratory where learners can explore superposition interactively instead of only mathematically.

It is designed especially for:

high school and undergraduate students
physics learners beginning quantum mechanics
researchers explaining qubit geometry visually
anyone interested in understanding quantum states intuitively
